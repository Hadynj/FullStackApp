# Full Stack App (Postgres, Express, React, Node.js: PERN)
The goal of this project is to connect a React app to a Postgres Database using an Express API with CRUD functions.

- [x] The React app can create, read, update, and delete data in the database.
- [x] The React app demonstrates the use of React components, hooks (useState) and props.
- [x] Final Submission:
      - Have a functional React frontend using Vite or any React toolchain set up to show the user an interface to perform CRUD operations
      - Have a functional Postgres database connected to an Express API
      - Connect React and Express API for frontend to backend data transfer 
      - Deploy the application to a cloud hosted platform (Heroku, Vercel, AWS, GCP, Microsoft Azure etc)
